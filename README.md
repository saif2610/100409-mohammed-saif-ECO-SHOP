## Project Overview

**EcoShop Impact** is a feature-rich, exam-ready Streamlit application that simulates an eco-conscious online shopping experience. It allows users to explore products, add them to a cart, visualize their environmental impact, and reflect on sustainability through data-driven insights and interactive visuals.

This version represents a **complete, polished implementation** suitable for **board exams, computer science practicals, project submissions, and demos**.

---

## Core Objectives

* Demonstrate how everyday shopping choices affect the environment
* Promote awareness of **carbon footprints and sustainability**
* Showcase advanced **Streamlit concepts** in a clean, professional format
* Combine UI design, data handling, and visualization in one cohesive app
